# Workspace Audit Scoring Rubric

## Score Dimensions

| Category | Max Points | Dimensions |
|----------|-----------|------------|
| SOUL.md | 50 | Clarity (10), Values (10), Boundaries (10), Voice (10), Token Efficiency (10) |
| IDENTITY.md | 30 | Completeness (10), Coherence (10), Engagement (10) |
| AGENTS.md | 50 | Memory Rules (10), Safety (10), Group Chat (10), Workflows (10), Subagent Independence (10) |
| Memory | 40 | Structure (10), Curation (10), Daily Logs (10), Promotion Pipeline (10) |
| Configuration | 30 | JSON Validity (10), Model Selection (10), Routing (10) |
| **Total** | **200** | |

## Rating Bands

- **0-60**: Needs Major Work
- **61-120**: Functional But Improvable
- **121-160**: Good
- **161-200**: Excellent

## Scoring Guidelines

### Score 0-2: Absent or Broken
The element is missing entirely, or present but fundamentally wrong.

### Score 3-5: Minimal
The element exists but is vague, generic, or incomplete.

### Score 6-8: Solid
The element is well-defined, specific, and functional. Minor improvements possible.

### Score 9-10: Excellent
The element is precise, opinionated, coherent, and production-ready.
